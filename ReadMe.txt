# Projeto Vinharia 

Este é um projeto desenvolvido para um site de uma vinharia fictícia. O site apresenta uma seleção de vinhos e oferece aos usuários uma experiência de compra online.

Descrição
O projeto Vinharia consiste em uma página da web onde os usuários podem visualizar uma seleção de vinhos disponíveis para compra. A página inclui recursos como:
Um popup de verificação de idade para garantir que os usuários sejam maiores de 18 anos.
Anúncio de frete grátis para compras acima de R$ 99,00.
Diversos banners promocionais.
Exibição de produtos com imagens, nomes, descrições e preços.
Possibilidade de adicionar produtos ao carrinho de compras.

## Funcionalidades

- Apresenta uma página inicial com informações sobre vinhos e destaque para promoções.
- Permite aos usuários navegar pelos produtos disponíveis.
- Possui um sistema de pop-up para verificar a idade do usuário antes de acessar o site.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Executar o Projeto

1. Clone o repositório para o seu ambiente de desenvolvimento.
2. Abra o arquivo `index.html` em um navegador da web.
